Create Video Thumbnails -:

              This API use to create video Thumbnails.
              METHOD -:     Post
              
SERVICE URL -:  https://vcdev.videocrypt.com/createThumbnail


            Input Parameter -:
            
 For this API you have to pass input parameter in body (Pass your data in raw).


            
  
            {
                "id": "xxxx_x_xxxxxxxxxxxxx",
                "Width": 1280,
                "Height": 720,
                "CallbackURL": "https://demo",
                "CodecSettings": {
                    "FrameCaptureSettings": {
                        "FramerateNumerator": 1, (No. of frame)
                        "FramerateDenominator": 5, (No. of Second)
                        "MaxCaptures": 5,
                        "Quality": 99 (maximum 100 quality)
                    }
                }
            }           
            

            
Header Parameter -:


            
 For this API you have to pass Header parameter in Header.


            
Note: Kindly encode your access and secret keys using a base64 encoder.) 


            
  
            accessKey:TTdBSlNPVVFRUs0RzJORDY5S===
            secretKey:WHc3QmhnS0VNUZBK1ZRQ29HWk1wc2M2UFMyYUltM2pmSFR5TGt===
            

           All parameters are required.
